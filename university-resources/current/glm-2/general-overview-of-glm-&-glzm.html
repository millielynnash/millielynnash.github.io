Overview of GLM Tests

GLM TestUse CaseKey AssumptionsSteps in SPSSInterpretation
ANOVACompares means across 2+ independent groups
- One-Way ANOVA (One factor)
- Two-Way ANOVA (Two factors)		- Independence of Observations
- Normality
- Homogeneity of Variance (Levene's Test)		Analyze > General Linear Model > UnivariateIf p ≤ .05, at least one group's mean differs significantly.

Post hoc tests (Tukey, Bonferroni) identify group differences.
ANCOVACompares means across groups while controlling for a covariate (e.g., age, IQ)- Same as ANOVA
- Linearity between Covariate and DV
- Homogeneity of Regression Slopes		Analyze > General Linear Model > Univariate

Add Covariate in 'Covariates' box		If p ≤ .05, groups differ after controlling for the Covariate. Covariate should significantly predict DV.
MANOVA (Multivariate Analysis of Variance)Compares means across multiple dependent variables simultaneously.- Independence of Observations
- Normality of each DV
- Homogeneity of Variance (Covariance matrices; Box's Test)		Analyze > General Linear Model > MultivariateIf p ≤ 0.05 (Wilks' Lambda, Pillai's Trace), at least one DV differs across groups.

Follow-up ANOVAs show which specific DV s.
MANCOVA (Multivariate Analysis of Covariance)Extends MANOVA by controlling for one or more covariates.- Same as MANOVA
- Linearity between Covariate and each DV		Analyze > General Linear Model > Multivariate

Add Covariate		If p ≤ 0.05, groups differ across multiple DV s after accounting for Covariate s.
Linear RegressionPredicts a Continuous DV from one or more continuous or Categorical Predictor s.- Linearity
- Independence of Error (Durbin-Watson)
- Normality of residuals
- Homoscedasticity		Analyze > Regression > LinearR2 shows how much variance is explained. p ≤ 0.05 for coefficients means predictors significantly affect DV.
Multiple RegressionPredicts a Continuous DV from multiple independent variables.- Same as Linear Regression
- No Multicollinearity (Variance Inflation Factor (VIF) < 10)		Analyze > Regression > Linear

Add multiple Predictor s		Check adjusted R2 for model fit.

Individual predictors are significant if p ≤ 0.05.
Logistic RegressionPredicts a binary (0/1) Dependent Variable- Linearity in log-odds
- No Multicollinearity		Analyze > Regression > Binary LogisticExp (B) (odds ratio) indicates the effect of predictors. p ≤ 0.05 means a significant predictor.
Repeated Measures ANOVATests for differences within the same subjects over time (e.g., pre-test, post-test)- Sphericity (Mauchly's Sphericity Test)
- Normality of Residuals		Analyze > General Linear Model > Repeated MeasuresIf p ≤ 0.05, at least one time point differs significantly.
Mixed Model (Repeated Measures & Between-Subjects)Examines both within-subject and between-subject effects (e.g., treatment effect over time across groups)- Same as Repeated Measures ANOVA
- Homogeneity of Variance		Analyze > General Linear Model > Repeated Measures

Add betwen-subject factors		If p ≤ 0.05, there's a significant interaction between time and group.

What is the Difference between a Covariate and Moderator

​Simple ContrastRepeated ContrastPolynomial Contrast
DefinitionCompares each level of the independent variable (IV) to a specific reference level (e.g., the first or last condition).Compares each level of independent variable to the previous level, analyzing changes step by step.Tests for trends across conditions, identifying whether the pattern is linear, quadratic, cubic, etc.
Comparison TypeEach level is compared to a fixed reference level.Each level is compared only to the level immediately before it.Examines overall patterns in the data rather than specific group comparisons.
UseUsed when there is a control group / baseline condition, and other groups are compared to it.Used when testing for gradual changes over time, treatment stages, or progressive conditions.Used when independent variable represents an ordered sequence (e.g., time points, increasing doses) and we want to test if the pattern follows a straight line or curve.
ExampleCompares B vs. A, C vs. A, and D vs. A (if A is the reference)

Alternatively, could compare B vs. D, C vs. D, etc. if D is the reference.		Compares B vs. A, C vs. B, and D vs. C.

Each contrast looks at differences between consecutive levels.		Linear Contrast:
- Tests if A > B > C follows a straight-line (increase or decrease)

Quadratic Contrast:
- Tests if values follow a U-shaped or inverted U-shaped pattern.

Cubic Contrast:
- Tests for more complex trends.
Advantages- Simple to interpret
- Works well when a specific group is at a logical baseline (e.g., placebo, pre-test, control group, etc.)		- Useful for detecting gradual progressions
- More sensitive to stepwise changes than ANOVA tests		- Can identify trends across multiple levels rather than just comparing individual groups.
- Use for dose-response studies, behavioral trends, etc.
Disadvantages- Does not show trends across levels (only compares each level to one reference).
- May miss patterns in the data if groups differ in more complex ways.		- Ignores non-adjacent comparisons (e.g., A vs. C or A vs. D).

- Not useful if differences between levels are not sequential.		- Harder to interpret than simple or repeated contrasts.
- May require quadratic/cubic/etc. terms to capture complex patterns.
Best Use- When comparing a treatment group to a control.
- When one condition serves as a logical starting or reference point.		- When assessing progressive changes over time (e.g., pre-, mid-, and post-test).
- When measuring stages of learning, recovery, treatment effects, etc.		- When analyzing dose-response relationships.
- When studying learning curves, reaction times, growth patterns, etc.
- When expecting non-linear trends in the data.
OutputProvides p-values and confidence intervals for each group vs. reference comparison.Provides p-values and confidence intervals for each stepwise change between levels.Provides p-values for each trend component (linear, quadratic, cubic, etc.) to determine which best fits the data.
