Parallel Mediation

Overview & Purpose

What is Parallel Mediation?

  • Parallel mediation involves multiple mediators that operate simultaneously. Each mediator is theorized to transmit the effect of the IV on the DV independently.

Purpose

  • To examine whether different processes (mediators) independently contribute to the effect of the IV on the DV. This model allows you to assess which mediator (s) play a significant role and to what extent.

Model Structure

  • IV → Mediator 1 → DV
  • IV → Mediator 2 → DV
  • (Mediators do not influence one another)

Key Assumptions & How to Check in SPSS

Linearity & Homoscedasticity

  • Generate scatterplots for DV vs. Each mediator.
  • Run separate regression analyses for each mediator (DV as outcome, mediator as predictor) and inspect residuals vs. Predicted plots.
    • Criteria: Residuals should show no systematic pattern.

Normality

  • Create histograms and Q–Q plots for the residuals of each regression.
    • Criteria: Skewness < |2.0| and kurtosis < |7.0|.

Mediator Independence

  • Run a correlation matrix (Analyze > Correlate > Bivariate) among the mediators.
    • Criteria: Inter-mediator correlations should be moderate (ideally, r < 0.80) to avoid redundancy.

Independence of Observations

  • As with simple mediation, verify study design and check the Durbin–Watson statistic (1.5–2.5).

Conducting Parallel Mediation in SPSS (Using PROCESS Macro)

  • Navigate to Analyze > Regression > PROCESS vX.
  • Use Model 4; PROCESS allows entering multiple mediators simultaneously.
  • Enter the IV, DV, and list all mediators.
    • Enable bootstrapping (e.g., 5,000 samples) to obtain confidence intervals for each indirect effect.
  • Click OK.

Interpreting the SPSS Output (Example Data)**

Indirect Effect via Mediator 1

  • Example: a 1 = 0.30, b 1 = 0.40 → Indirect effect = 0.30 × 0.40 = 0.12
    • Bootstrapped 95% CI: [0.03, 0.22] (significant if CI does not include zero, p = .012).

Indirect Effect via Mediator 2

  • Example: a 2 = 0.25, b 2 = 0.35 → Indirect effect = 0.25 × 0.35 = 0.0875
    • Bootstrapped 95% CI: [0.01, 0.18] (significant, p = .025).

Total Indirect Effect

  • Sum of individual indirect effects (if both are significant).

Direct Effect (IV > DV Controlling for Mediators)

  • Example: c′ = 0.30, SE = 0.08, p < .001

Example APA Write-Up for Parallel Mediation

  • A parallel mediation analysis was conducted using the PROCESS macro (Model 4) with 5,000 bootstrap samples to evaluate whether [Mediator 1] and [Mediator 2] independently mediated the relationship between [IV] and [DV]. The indirect effect through [Mediator 1] was significant (b = 0.12, SE = 0.06, 95% CI [0.03, 0.22], p = .012), as was the indirect effect through [Mediator 2] (b = 0.09, SE = 0.05, 95% CI [0.01, 0.18], p = .025). The direct effect of [IV] on [DV] remained significant (b = 0.30, SE = 0.08, p < .001), suggesting that both mediators contributed uniquely to the overall relationship.”
