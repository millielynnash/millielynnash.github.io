Simple Mediation

Overview & Purpose

What is Mediation?

  • Mediation examines how the effect of an independent variable (IV) on a dependent variable (DV) is transmitted through a third variable, called the mediator (M). In simple mediation, there is one mediator.

Purpose

  • To determine if the IV influences the DV directly and/or indirectly via the mediator. The model splits the total effect into
Direct Effect (c')
  • The portion of the effect of IV on DV that is not explained by the mediator.
Indirect Effect (a x b)
  • The portion of the effect that is transmitted from IV to DV through the mediator.

Model Structure

Key Assumptions & How to Check in SPSS

Linearity

  • Relationships among IV, mediator, and DV are linear.
How to Check
  • Scatterplots:
    • Go to Graphs > Legacy Dialogs > Scatter/Dot and plot DV vs. IV and DV vs. M.
      • Look for a straight-line pattern.
    • Residuals vs. Predicted Plot:
      • Run Analyze > Regression > Linear…; click Plots… to assign standardized predicted values (X-axis) and standardized residuals (Y-axis).
      • Criteria: No systematic pattern (i.e., randomness, with any curve test yielding p > .05).

Homoscedasticity

  • Residuals have constant variance across all levels of predictors.
How to Check
  • Examine the same residuals vs. predicted plot for an even spread (no funnel shape).

Normality

  • Residuals are approximately normally distributed.
How to Check
  • Create histograms and Q–Q plots using Graphs > Legacy Dialogs > Histogram or Q–Q plot options.
    • Criteria: Skewness values less than |2.0| and kurtosis values less than |7.0|; Q–Q plot points should lie near the diagonal line.

Independence

  • Observations are independent.
How to Check
  • Confirm via your study design.
  • Review the Durbin–Watson statistic from your regression output (values between 1.5 and 2.5 are acceptable).

No Omitted Confounders

  • There are no unmeasured variables that confound the relationship between the mediator and DV.
How to Check
  • This is primarily a design issue; however, consider including control variables if needed.

Conducting Simple Mediation in SPSS (Using PROCESS Macro)

Run the Analysis
  • Go to Analyze > Regression > PROCESS vX by Andrew Hayes
  • Choose Model 4 (designed for simple mediation).
  • Set the DV, IV, and mediator in the appropriate boxes.
  • Enable bootstrapping (commonly set to 5,000 samples) to obtain robust confidence intervals (CIs) for the indirect effect.
  • Click OK.

Interpreting the SPSS Output (Example Data)

  • Direct Path (IV → M):
    • Example: a = 0.40, SE = 0.10, p = .001
  • Path from Mediator to DV (M → DV, controlling for IV):
    • Example: b = 0.50, SE = 0.12, p = .002
  • Indirect Effect (a × b):
    • Example: 0.40 × 0.50 = 0.200.40 \times 0.50 = 0.20
      • Bootstrapped 95% CI: [0.08, 0.35] (does not include zero → significant mediation).
  • Direct Effect (IV → DV controlling for M):
    Example: c′ = 0.25, SE = 0.09, p = .005
  • Total Effect (IV → DV without mediator):
    Example: c = 0.45, SE = 0.10, p < .001

Example APA Write-Up for Simple Mediation

  • A mediation analysis was conducted using the PROCESS macro (Model 4) with 5,000 bootstrap samples to examine whether [Mediator] mediated the relationship between [IV] and [DV]. The total effect of [IV] on [DV] was significant (b = 0.45, SE = 0.10, p < .001). When [Mediator] was entered into the model, the direct effect decreased (b = 0.25, SE = 0.09, p = .005). The indirect effect was significant (b = 0.20, SE = 0.07, 95% CI [0.08, 0.35]), indicating that [Mediator] partially mediated the relationship between [IV] and [DV].

Parallel Mediation

Overview & Purpose

What is Parallel Mediation?

  • Parallel mediation involves multiple mediators that operate simultaneously. Each mediator is theorized to transmit the effect of the IV on the DV independently.

Purpose

  • To examine whether different processes (mediators) independently contribute to the effect of the IV on the DV. This model allows you to assess which mediator(s) play a significant role and to what extent.

Model Structure

  • IV → Mediator 1 → DV
  • IV → Mediator 2 → DV
  • (Mediators do not influence one another)

Key Assumptions & How to Check in SPSS

Linearity & Homoscedasticity

  • Generate scatterplots for DV vs. each mediator.
  • Run separate regression analyses for each mediator (DV as outcome, mediator as predictor) and inspect residuals vs. predicted plots.
    • Criteria: Residuals should show no systematic pattern.

Normality

  • Create histograms and Q–Q plots for the residuals of each regression.
    • Criteria: Skewness < |2.0| and kurtosis < |7.0|.

Mediator Independence

  • Run a correlation matrix (Analyze > Correlate > Bivariate) among the mediators.
    • Criteria: Inter-mediator correlations should be moderate (ideally, r < 0.80) to avoid redundancy.

Independence of Observations

  • As with simple mediation, verify study design and check the Durbin–Watson statistic (1.5–2.5).

Conducting Parallel Mediation in SPSS (Using PROCESS Macro)

  • Navigate to Analyze > Regression > PROCESS vX.
  • Use Model 4; PROCESS allows entering multiple mediators simultaneously.
  • Enter the IV, DV, and list all mediators.
    • Enable bootstrapping (e.g., 5,000 samples) to obtain confidence intervals for each indirect effect.
  • Click OK.

Interpreting the SPSS Output (Example Data)**

Indirect Effect via Mediator 1

  • Example: a1 = 0.30, b1 = 0.40 → Indirect effect = 0.30 × 0.40 = 0.12
    • Bootstrapped 95% CI: [0.03, 0.22] (significant if CI does not include zero, p = .012).

Indirect Effect via Mediator 2

  • Example: a2 = 0.25, b2 = 0.35 → Indirect effect = 0.25 × 0.35 = 0.0875
    • Bootstrapped 95% CI: [0.01, 0.18] (significant, p = .025).

Total Indirect Effect

  • Sum of individual indirect effects (if both are significant).

Direct Effect (IV > DV Controlling for Mediators)

  • Example: c′ = 0.30, SE = 0.08, p < .001

Example APA Write-Up for Parallel Mediation

  • A parallel mediation analysis was conducted using the PROCESS macro (Model 4) with 5,000 bootstrap samples to evaluate whether [Mediator 1] and [Mediator 2] independently mediated the relationship between [IV] and [DV]. The indirect effect through [Mediator 1] was significant (b = 0.12, SE = 0.06, 95% CI [0.03, 0.22], p = .012), as was the indirect effect through [Mediator 2] (b = 0.09, SE = 0.05, 95% CI [0.01, 0.18], p = .025). The direct effect of [IV] on [DV] remained significant (b = 0.30, SE = 0.08, p < .001), suggesting that both mediators contributed uniquely to the overall relationship.”

Serial Mediation

Overview & Purpose

What is Serial Mediation?

  • Serial mediation (also called chain mediation) involves two or more mediators arranged in a specific sequential order. The IV affects the first mediator, which in turn affects the second mediator, and so on, eventually influencing the DV.

Purpose

  • To investigate the process by which the IV impacts the DV through a chain of mediators, highlighting the step-by-step mechanism and temporal ordering of effects.

Key Assumptions & How to Check in SPSS

Linearity

  • Generate scatterplots and residual vs. predicted plots for each link: IV → M1, M1 → M2, and M2 → DV.
  • Use histograms and Q–Q plots for residuals in each regression.
    • Criteria: As before, look for random dispersion, skewness < |2.0|, kurtosis < |7.0|, and Durbin–Watson between 1.5 and 2.5.

Sequential (Causal) Ordering

  • There must be theoretical or empirical support for the hypothesized order (e.g., M1 influences M2).
    • Run intermediate regressions (e.g., regress M2 on M1) and ensure that coefficients are significant (p < .05) in line with your theory.

No Omitted Confounders in Sequence

  • Consider and control for variables that may confound the relationships between mediators if necessary.

Conducting Serial Mediation in SPSS (Using PROCESS Macro)

  • Go to Analyze > Regression > PROCESS vX.
  • Choose Model 6 (designed for serial mediation).
  • Enter the IV, DV, and mediators in the specific sequential order.
  • Click Options:
    • Enable bootstrapping (e.g., 5,000 samples) to obtain confidence intervals.
  • Click OK.

Interpreting the SPSS Output (Example Data)

  • Path IV → Mediator 1:
    • Example: a1 = 0.35, SE = 0.09, p = .001
  • Path Mediator 1 → Mediator 2:
    • Example: d = 0.45, SE = 0.10, p = .001
  • Path Mediator 2 → DV (controlling for IV and M1):
    Example: b2 = 0.30, SE = 0.08, p = .002
  • Sequential Indirect Effect (IV → M1 → M2 → DV):
    Calculation: a1 × d × b2 = 0.35 × 0.45 × 0.30 = 0.0473
    • Bootstrapped 95% CI: [0.01, 0.10] (if CI does not include zero, the effect is significant, e.g., p = .018).
  • Direct Effect (IV → DV controlling for mediators):
    Example: c′ = 0.35, SE = 0.09, p < .001

Example APA Write-Up for Serial Mediation

  • A serial mediation analysis was conducted using the PROCESS macro (Model 6) with 5,000 bootstrap samples to test whether [Mediator 1] and [Mediator 2] sequentially mediated the relationship between [IV] and [DV]. The analysis indicated that [IV] significantly predicted [Mediator 1] (b = 0.35, SE = 0.09, p = .001), and [Mediator 1] significantly predicted [Mediator 2] (b = 0.45, SE = 0.10, p = .001). Furthermore, [Mediator 2] significantly predicted [DV] (b = 0.30, SE = 0.08, p = .002). The sequential indirect effect was significant (b = 0.047, SE = 0.02, 95% CI [0.01, 0.10], p = .018), indicating that the effect of [IV] on [DV] was transmitted via [Mediator 1] and [Mediator 2] in sequence. The direct effect of [IV] on [DV] remained significant (b = 0.35, SE = 0.09, p < .001), suggesting partial mediation.

Summary of Key Steps

  1. Check Assumptions in SPSS:
    • Generate scatterplots, residual vs. predicted plots, histograms, and Q–Q plots.
    • Verify diagnostic values: skewness (< |2.0|), kurtosis (< |7.0|), Durbin–Watson (1.5–2.5), and inter-mediator correlations (r < 0.80 for parallel mediation).
    • For serial mediation, confirm intermediate regression steps are significant (p < .05).
  2. Conduct Mediation Analysis with PROCESS:
    • Use the appropriate model: Model 4 for simple and parallel mediation; Model 6 for serial mediation.
    • Enter variables as specified and enable bootstrapping (commonly 5,000 samples) to generate 95% confidence intervals.
  3. Interpret Output:
    • Evaluate p-values for each path (significant if p < .05).
    • For indirect effects, confirm that bootstrapped confidence intervals do not include zero.
