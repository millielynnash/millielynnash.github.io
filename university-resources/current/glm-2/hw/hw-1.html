Please Describe the Difference between Descriptive and Inferential Statistics.

  • Descriptive statistics describe the state of the data. It summarizes and describes the characteristics of a data set whereas inferential statistics infer information about the sample participants based on their data. Inferential statistics uses data from a sample to make predictions or draw conclusions about a larger population. The results allow us to analyze the sample and infer predictions about the larger population.

Please Describe the Difference between the General Linear Model (GLM) and Generalized Linear Model (GLzM).

  • The General Linear Model (GLM)
    • Assumes the residuals (variation of variables) will be 'normally distributed'
  • Betas are regression weights that are optimally estimated based on their contribution of each variable to reasonable predictions of Y.
  • Absolute value of b depends on the metric scale of the x variable that they modify
  • The Generalized Linear Model (GLzM)
    • Extension of GLM
    • Outcome variable follows different distributions than the 'normal distribution' (i.e., probability distributions, binomial distributions, gamma distributions, etc.)
    • Binary/dichotomous, count data, etc.
    • Outcome variable is traditionally related to the linear model through link functions.

Please Describe Each of the Four Levels of Measurement in Statistics and Provide Examples of Variables at Each Level.

Nominal Variable

  • Categories that are distinct and which do not have numerical value
  • Examples
    • School
    • Eye color
    • Marital status

Ordinal Variables

  • Numerical values that only signify ranking.
  • Examples
    • Risk level
    • Likert-scale questions

Interval Variable

  • Numerical data that has equibalent ranges between values but no meaningful zero.
  • Examples
    • SAT scores
    • Percentile rankings
    • Temperature in Fahrenheit or Celsius

Ratio Variables

  • Same as interval, but with a meaningful zero.
  • Data can be ranked, intervals are consistent.
  • Examples
    • Height
    • Age
    • Weight
    • Temperature in Kelvin
