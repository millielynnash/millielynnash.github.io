Please Describe the Difference between Unstandardized and Standardized Coefficients.

Unstandardized Coefficients (Raw)

  • The change in the dependent (outcome) variable for a one unit change in the independent (predictor) variable, while holding all other variables constant.
  • Expressed in the original units of the dependent (outcome) and independent (predictor) variables.

Standardized Coefficients (Beta)

  • The values obtained after standardizing all variables (both dependent and independent variables) to have a mean of 0 and a standard deviation of 1.
  • This allows for the newly calculated variables to represent the change in the dependent variable (in standard deviations) for one-standard deviation change in the value of the independent variable.

Please Describe Zero-Order Correlations, Partial Correlations, and Semi-Partial Correlations.

Zero-Order Correlations

  • Measures the relationship between two variables without controlling for any other variable(s) using simple correlation.
  • We would use this when we want to understand the simple bivariate relationship between two variables.
  • We would likely use the Pearson Correlation Coefficient R for this type of analysis.

Partial Correlations

  • Measures the correlation between x and y while controlling for x2 (on both x and y).
  • We would likely use this when we want to understand the direct relationship between two variables, independent of influence of other variables.

Semi-Partial Correlations

  • Measures the relationship between two variables while controlling for the effect of one or more variables on only one of the two variables.
  • Correlation between x and y while controlling for x2 (on x1 only)
  • We would likely use this when we want to understand the exact contribution of an independent (predictor) variable to the dependent (outcome) variable, after taking into account the influence of other variables on the independent (predictor) variable.

Please Describe the Differences between Simultaneous, Hierarchical, and Stepwise Regression.

Simultaneous Multiple Regression

  • All predictors are added into the regression model at the same time.
  • The model evaluates the contribution of each predictor to the outcome variable while controlling for all other predictors.
  • R-squared is a static value that represents the variance explained in the outcome variable by the collective set of predictors.
  • Continuous variable is added as it is
  • Nominal Variable must be dummy coded
  • Variables with non-zero values should be ‘centered’

Hierarchical Multiple Regression

  • Predictors are entered into the model in a pre-specified order based theory-driven blocks.
  • The model is built in blocks (steps) with each block adding one or more predictors. The change in the model’s R-squared is assessed at every step.
    • A series of multiple-regression are performed, with each block being added to the next until all variables are included in the model.
    • The final block of hierarchical multiple regression = simultaneous multiple regression.
  • Very useful to look at the change in R-squared for each variable or blocks of variables
    • Allow for testing hypotheses in the order of importance/priority of predictors.
  • Squared semi-partial correlation provides the unique variance explained by a given variable.

Stepwise Multiple Regression

  • Predictors are added/removed from the model based on statistical criteria such as p-values or R-squared change rather than being theory-driven. Predictors are added/removed in data-driven blocks.
  • Mathematically derives the highest R-squared value with the least number of predictors.
  • CAUTION: NEVER USE THIS METHOD UNLESS STUDY IS ABSOLUTELY EXPLORATORY.

Forward Stepwise Regression

  • Most predictive variables are added first

Backward Stepwise Regression

  • Least predictive variables are removed first

What is Yielded When You Square a Semi-Partial Correlation?

  • When you square the semi-partial correlation, it yields the proportion of the variance of y uniquely explained by x while controlling for all other predictors.

Please Define and Describe the Four Primary Assumptions of Regression

MLR in SPSS

  1. Analyze > Regression > Linear
  2. Move dependent variable to Dependent. Move independent variable(s) to the Independent(s).
    - In Statistics, select Estimates (under Regression Coefficients), Model Fit, Descriptives, and Durbin-Watson (under Residuals, independence of errors). Click Continue.
    - In Plots, select Histograms and Normal Probability Plot (under Standardized Residual Plots). Move ZPRED under Y. Move ZRESID to X. Click Continue.
    - In Save, select Cook’s (under Distances). Click Continue.
    3.     Click OK.

Linearity

  • The relationship between the independent (predictor) variables and the dependent (outcome) variables is linear.
  • How to Check
  1. Use Scatterplots to visualize relationships. Click Graphs > Legacy Dialogs > Scatter/Dot.
  2. Click Simple Scatter. Click Define.
  3. Move your dependent variable (outcome) to the Y-Axis. Move your independent variable (predictor) to the X-Axis.
  4. If the scatterplot is approximately linear, the assumption is met.
    • If not linear, apply transformation.

Independence of Error

  • Observations are independent of each other (no autocorrelation, where observations are NOT independent)
  • How to Check
  1. Use the Durbin-Watson test by clicking the option under Linear Regression: Statistics.
  2. If the Durbin-Watson test falls within the range of 1.5 – 2.5, the assumption is met.
    • If data is autocorrelated, use alternative models such as the generalized least squares (GLS) model.

Homoscedasticity

  • The residuals (errors) have constant variance across all levels of independent variable(s).
  • How to Check
  1. Plot residuals vs. predicted values and look for a random scatter.
  2. In SPSS, request Linear Regression Plots. Move Y > ZPRED. Move X > ZRESID.
    • If the scatterplots shows no overall pattern, the assumption is met.
    • If assumption is violated, you can transform the dependent variable, use standard errors, or apply weighted least squares (WLS).

Normality

  • The residuals (errors) are normally distributed.
  • How to Check (For None or One Grouping Variable)
  1. Click Analyze > Descriptives > Explore
  2. Transfer the variable that needs to be tested for normality into the Dependent List box.
  3. Click on Statistics. Leave Descriptives selected.
  4. Click the Plots. Select Factor Levels Together under Boxplots. Select Stem-and-Leaf under Descriptive. Select Normality Plots with Tests. Click Continue.
  5. Click OK.
  6. Check Shapiro-Wilk Test of Normality, Normal P-P Plot, Histogram, and Skewness/Kurtosis.
    • If the Sig. value of the Shapiro-Wilk Test is greater than p = .05, the data is normal.
      • If it is below .05, the data significantly deviate from a normal distribution.
    • If the points are generally along the diagonal of the Normal P-P Plot, the assumption is met.
    • If the Histogram is approximately normal (distribution curve is bell-shaped), the assumption is met.
    • If Skewness and Kurtosis are between +2 and -2, the assumption is met.

Please Define and Describe Distance, Leverage, and Influence

Distance

  • How far a case is from some point of reference

Mahalonobis' Distance

  • Identifies the distance of cases from the mean (s) of predictor (s). Extreme cases in multivariate space are found via a cut-off point (critical value of Chi-Square Test)

Standardized Residuals

  • Measure the distance between an observed value and its predicted value that have all been converted to z-scores.
  • Typically flag cases if they are 2 or 3 standard deviations away from the mean.

Leverage

  • How far a point deviates from the mean of that variable.
  • Gauges the influence of the observed values over the predicted values.
    • Ranges from zero to (N – 1) / N
    • Average value is (k + 1) / N
    • No greater than 2 (k + 1) / N or 3 (k + 1) / N

Influence

  • The degree to which a variable can ‘pull’ the regression towards itself.
  • Cook's Distance
    • Measures the overall influence of a case on the model
    • Typically flagged if value is over 1

What is Meant by a Factor Loading in Exploratory Factor Analysis (EFA)?

  • Factor loadings help determine which variable is associated with which latent factor. For example, several likert-scale questions on a questionnaire observing psychological outcomes may load highly on a ‘depressive/depression’ factor, showing they measure the same underlying construct. Higher values of factor loadings (e.g., > .4 or .5) indicate that a variable is a strong indicator of that factor. Lower values suggest the variable is less relevant to that factor. EFA can allow us to group variables with higher factor loadings on the same factor, reducing the complexity of our data.
