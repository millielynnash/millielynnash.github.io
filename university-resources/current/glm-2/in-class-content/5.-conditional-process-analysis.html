5. Conditional Process Analysis

  • Disordinal (when the moderation slopes cross)

Lecture

Overview of Moderation, Mediation, and Conditional Process Analysis

Interaction = Moderation

  • An interaction/moderation occurs when the effect of an independent variable (x) on an Outcome variable (y) depends on the values of another independent variable (w).

  • Controlling for the other predictors in a model, does x predict y?
    • The above is logical only if the relation between x and y does not depend on another predictor in the model.
    • If a relation does depend on a moderating variable (w), then moderation has occurred and the relations need to be explored.
  • Moderation can occur between categorical predictors, categorical and one continuous predictor, or between continuous predictors.
  • How you decide if its a moderator?
    • If x and y AND w and y we know are related, we could include in the moderation.
Types of Interactions
  • Categorical & Categorical
  • Continuous & Categorical
  • Continuous & Continuous
Consequences of Ignored Moderation(s)
  • Model will be misspecified
  • Inflates the MSE residuals
  • F-test will be invalid (F = MSR / MSE)
Mean Centering
  • Mathematical transformation of variables to center around the mean instead of zero (e.g., making the mean equal zero)
  • Subtract the mean of a variable from all observations for that variable.
  • Increases interpretability and decreases Multicollinearity
  • Changes the interpretation of the intercept
    • The value of the dependent variable when the independent variables are at their averages rather than zero.
  • Debated in the literature, so choose for yourself what makes the most sense. Just report it correctly!
    • Pro mean centering (Aiken & West, 1991)
    • Against mean centering (Hayes, 2018)

Mediation (Indirect Effects)

  • A mediation (indirect effect) occurs when an Independent Variable (x) relates to an outcome variable (y) through a mediating variable (m).
  • a = first-stage relation (effect of x on m)
  • b = Second-stage relation (effect of m on y)
  • c = total effect (e.g., regression/correlation coefficient)
  • c' = Direct effect (effect of x on y controlling for m)
  • Indirect effect = a x b (a multiplied by b)
  • The effect of x on y does not need to be significant for a mediation to be possible (e.g., x significantly relates to m, which significantly relates to y, but x does not predict y directly)
  • Multiple mediators may be present operating either in tandem (Parallel Mediation) or in sequence (Serial Mediation)
    • Parallel Mediation
      • Controls for two covariates
    • Serial Mediation
    • It is possible that x and y are not direct (partial or complete mediation)

Conditional Process Analysis

  • The examination of both moderation and mediation processes occurring at the same time.
  • Mediations can be moderated
  • Index of mediation = a value of the distance between two indirect effects
    • If significant, evidence of moderated mediation.
    • If not significant, mediation is not moderated.

PROCESS Macro Setup, Syntax, and Interpretation

A Note on Dummy Coding

  • If you want to replicate the findings of an ANOVA via Regression exactly, be sure to code your Nominal Variable s around zero (-1, 1, or -. 05, .05) to achieve Main Effects Parameterization. This yields coefficients that represent the main effects of predictor variables.
  • Using Simple Effects Parameterization (0, 1) yields the same interpretation as dummy coding. The effect of x when w = 0, or the effect of w when x = 0.
  • Which one to choose is a matter of preference and what research question (s) and hypotheses you are trying to address.

PROCESS Macro Setup

  • A tool developed by Andrew Hayes to easily run Conditional Process Analysis
  • Download files at (Download - The PROCESS macro for SPSS, SAS, and R) or from our Carmen site
  • Extract all files from the zipped folder and move files to somewhere you can easily access

PROCESS Syntax Structure

  • Process y=outcome/x=independent/m=mediators/w=moderator/z-second moderator/cov=covariate list/model number from Hayes text/seed = any number you want that locks the random number generator.
  • PROCESS includes 5000-sample bootstrapped confidence intervals by default
  • Conf = 99 increases to 99% confidence interval, which is more conservative. Useful for biased estimates, non-normal data, Multicollinearity, etc.
  • You MUST include seed = some number so that the findings remain the same. If the random number generator is allowed to freely draw, the findings will be different every time.

PROCESS Output Interpretation

  • For moderation
  • For mediation
  • For Conditional Process Analysis

Lab

Simple Moderation Analysis in SPSS

Example

  • Scenario
    • We want to find out whether the effect of mom's IQ on kid's score on a cognitive test is impacted by mom's high school status.
    • In simple terms, do mother's high school status moderate the relationship between mother's IQ and kid's performance on a cognitive test?
Diagram
  • Model 1
Steps in SPSS
  • Analyze -> Regression -> PROCESS v 4.2 by Andrew F. Hayes
    - Note: Make sure each variable name is not more than 8 characters
  • Drag the Dependent Variable into the 'Y variable' box (kid_score)
  • Drag the Independent Variable to the 'X variable' box (mom_iq)
  • If you have any covariate(s), move it to the covariate(s) box
  • Drag the Moderator to the 'moderator variable w' box (mom_hs)
  • Remember not to use the 'Paste' button!
    - If you want to use the syntax, you will have to generate the syntax by yourself.
  • If your Moderator or Independent Variable (x) is Categorical, click on 'Multicategorical' and select the coding scheme of your choice.
  • A variable specified as 'Multicategorical' must have at least 3 categories.
  • You don't have to manually recode your Nominal Variable s.
Using the Syntax

Process y = kidscore/ x=mom_iq/ w=mom_hs/ model=1/ center=2/ plot=1.
Code Generated for the Graph

DATA LIST FREE/

GRAPH/SCATTERPLOT= mom_iq WITH kidscore BY mom_hs.
Output

  • You will have to copy the 'DATA LIST FREE/' and to the period and paste into your syntax window
Graph

  • Although there is a positive effect of mom_iq on kidscore, the magnitude of the effect (the slope) is higher for children whose parents did not graduate high school (the blue line).

Moderation in PROCESS (Continued)

Scenario Problem

  • Let's say that we have a theory that the more time you spend with a puppy, the happier you are. Also, the amount of love you have for puppies will moderate that effect.
  • We will use the puppy_love dataset to run this scenario in PROCESS, but there are a few things to note:
    • Make sure all variables you are using are 8 characters or less in length
    • We will also set up the problem to create a plot
    • We are interested in any points of W (the moderator) where the significance changes. This will be determined by the Johnson-Neyman approach.

In SPSS

  • Go to Analyze > Regression > PROCESS
  • Move over your y, x, and w variables.
  • Typically, we would change the model #, but given that we are working with Model 1, we are keeping it the default of 1.
  • Click on 'Options'
  • This is the Options pop-up.
  • To generate the code for making your plot, select Generate code for visualizing interactions.
  • To get Johnson-Newman results, click on the box that is circled below for 'Johnson-Newman Output'
  • Click Continue then OK
Using Syntax

Process y = Happy/ x= Dose/w=puplove/model=1/center=0/jn=1/plot=1/moments=0.

JN = 1 or JN = 0
  • Jn=1
    • We want to get the Johnson-Newman output, if any.
  • Jn=0
    • Omitting it from the syntax means we do not want the J-N output.
Plot = 1 or Plot = 0
  • Plot=1
    • We want the syntax for the plot to be provided to us.
  • Plot=0
    • Omitting it from the syntax means we do not want the syntax.
Which Model
  • Model=1
    • The simplest model where we have one moderator moderating the relationship between the focal predictor and the outcome variable.
  • Please check the 'Appendix A' document for Process Macro to learn more about other models and their numbers.
Moments = 1 or Moments = 0
  • Moments=1
    • We want to create three categories of the moderator by using the mean, 1 SD above the mean, and 1 SD units below the mean (the most common approach).
    • (NB: only do this when your moderator is a continuous variable)
    • This is done to help with interpretation
  • Moments=0
    • Omitting it from the syntax means we are creating three categories of the moderator by using the 16th percentile, 50th percentile, and the 84th percentile. This is the default.
  • If your moderator is a categorical with more than 2 categories, you can add /MCW=1 to the syntax to automatically create dummy variables for you. Check the appendix file for other coding schemes and their corresponding numbers.
Center = 0, Center = 1, Center = 2
  • Center=0
    • Omitting it from the syntax
    • Means NO CENTERING
  • Center=1
    • All variables that define products should be automatically mean-centered
  • Center=2
    • Only the continuous variables that defines products should be automatically mean-centered.
  • Recommendation, if you are thinking about mean-centering, then:
    • Choose Center=2, if the moderator or the focal predictor is a continuous variable.
    • Choose Center=1 if both the moderator or the focal predictor are a continuous variables.
    • Choose Center=0 if both the moderator or the focal predictor are a categorical variable

Reading the Output

  • This first part of the output shows us the model we built including the model number, our N, and the model summary.
  • Model summary includes the R2 value, which in this case shows a significant amount of variance explained (37.63%)
Model Coefficients
  • Below, in the 'Model' table, you can see the coefficients for the intercept as well as for the slopes of Dose, Love of Puppies, and the automatically created interaction term.
  • Just below that you see that the interaction term is dose x puppylove
Conditional Effects of the Focal Predictor
  • This next significant table shows us the conditional effects of W at the 16 th, 50 th, and 84 th percentiles.
  • Here, we see that when puppy love is at the 16 th percentile (puppy love = 1), the effect is 1.81 (p = .0013). Likewise, when puppy love is at the 50 th percentile (2.5), the effect is also significant (p = .0046) at 1.1694.
  • However, the effect is no longer significant at the 84 th percentile for puppy love, (p > .05).
Johnson-Newman Table
  • The next table is the Johnson-Newman results.
  • The Johnson-Newman approach looks for the values of W, where the p-values of the conditional effect changes to above/below the chosen significance level (. 05). So, it looks for the values of W that cause t-crit value to equal zero for when determining the conditional effect of X, where the p-value is exactly the alpha, or .05 in most cases.
  • At what values of W, do the results go from significant to non-significant, or vice-versa, in terms of conditional effect?
  • Your results can either be of one, two, or zero values.
    • If there is one value, that means there is only one value of W where the conditional effect either becomes significant or loses significance.
    • If there are two values, that means there are two points at which W will change in significance. This will either mean the middle range is significant, but not the high and low values of W, or that the high and low values of W are significant, but not the middle range. The middle range is determined by the W values themselves.
    • If there are no values, then the conditional effect is either always significant or never significant for all values of W.
Graphical Representations
  • Next are graphical representations for what different types of Johnson-Newman results may look like.
  • Where the line cross .05, that's where this approach looks for the value of W, giving us more insight into what extent our moderator... Well, moderates.
  • Graphs for two or no solutions to JN approach

  • See that the Johnson-Newman technique gave us only one value, 3.3364.
  • By finding the value in the table below, we can see that the conditional effect is significant before W = 3.3364, but non-significant after that point. So, it is significant with lower levels of puppy love. (First column, row 3.3364)
Interpretation of the Johnson-Neyman Output
  • The moderator (puplove) has significant impact on the relationship between time spent with Puppies and Happiness, only at its low values (i.e., from 0.00 to 3.3364)
    • This implies that, the less an individual loves a puppy, the more time spent with puppy affect (increases) their happiness.
  • At its higher values, the moderator (puplove) does not have a significant effect on the relationship between time spent with Puppies (focal predictor) and Happiness (outcome variable)
    • This implies that, increasing the time spent with Puppy does not affect/change (increases) happiness for individuals who already love Puppies.

How to Generate a Plot

  • The next part of the table give us the syntax to generate the plot
  • Double click on the matrix table and it will allow you to highlight and copy parts of the text.
  • Highlight and copy from the DATA LIST FREE/ until the period at the end.
Creating the Base Plot
  • Once you copy the code, paste it into a new syntax file.
    • There is nothing you need to change. Just highlight and play the syntax.
  • The results will look like the graph below, a plot with points but no lines.
Creating Plot Lines
  • To create plot lines, double click on the plot itself. A 'Chart Editor' pop-up will appear.
  • Click on the Add lines at Subgroups button (It looks like a scatterplots with TWO lines through it, not one) (circled in red)
  • You will get the properties pop-up now, but by default, it will give you the lines you need, as you can see now in the red boxed area.
  • Just close the properties box and the chart editor box.
  • This is what your chart should look like now. You can now have lines that can help you to visualize the changes in X vs. Y, given the subgroups of W.

Multicategorical Moderator 'W'

Scenario Problem

  • Pretend that literature shows that the more someone loves puppies, the happier they are. We would like to test that theory, but with a moderator of time spent with puppies during a trial (dose).
  • Here, we are recognizing that Dose is a multi-categorical variable for control group (0 minutes, 15 minutes, or 30 minutes with puppies).
  • Typically, we would need to dummy-code these variables, but PROCESS does it for you, provided you remember one extra step.

To Make 'W' Multi-Categorical in PROCESS

  • First, set up the model as usual in PROCESS.
    • Note. Where the variables are in the image based on the new hypothesis.
  • Next click on 'Multicategorical'
  • The next pop-up you will get is the multicategorical box.
  • Here, you will select the checkbox under 'Variable W' to indicate that your moderator is multicategorical.
    • Note. The other options, indicating that you can also do this for any X or Z variables(s) that have 3 or more categories.
  • Click Continue.
  • Click OK.

Changes in the Output to Note

  • One of the changes is the dummy coding. The coding scheme is provided just below the model overview and will look something like the table below.
  • The next change is the model and product terms key tables. Just keep in mind the W coding scheme from before when interpreting each line, since now we are recognizing the separate levels of our W.
  • The final change is the set-up of the conditional effects table.
  • Usually, this will either be representative of the 16 th, 50 th, and 84 th percentile (default), or the means +/- 1 SD (if selected). However, now since we are recognizing W as a categorical variable, the doses are listed as the separate categories.
    - Note how there is no mention of the percentiles in the 'Analysis Notes and Errors' section.
