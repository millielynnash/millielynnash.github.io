Quick Review

  • Regression Goal:
    • Use IV (x) to explain the variability (variance) in DV (y)
  • Partition Variance:
    • SSTO = SSR + SSE
  • Model Summary (Variance Explained):
    • R2 = SSR ÷ SSTO
      • In SLR:
        • R2 = rxy2
  • ANOVA table (omnibus test):
    • Is it worth using the regression model to explain y?
    • Fobs = MSR ÷ MSE = (SSR ÷ 1) ÷ (SSE ÷ (n-2)) (the df here are for SLR only)
    • Reject H0 if p < 𝛼 (usually .05), indicating significant model.
    • Report omnibus test statistics:
      • F (1, n-2), p-value (df for SLR only)
  • Optional regression coefficient estimates (b0 and b1) were found by the ordinary least squares (OLS) estimate. For SLR:
    • Intercept (Constance):
      • b0 = y-bar - b1(x-bar)
      • Regression line goes through both x and y means.
    • Slope:
      • b1 = rxy (sy÷sx)
      • Related to the Pearson correlation between x and y
      • Same scale as y per x change

Two Ways to Write a Regression Equation (SLR)

  • y-hat = b0 + b1xi
    • Describing the regression line
    • y-hat is the fitted (i.e., expected/predicted) value (the line)
  • y-hat = b0 + b1xi + ei
    • Describing data points
    • ei (𝜀i for population) is the deviation between the data point and the fitted line (i.e., error, residual) for subject i
  • Regression coefficients are available from the last table in SPSS output.

Coefficient 𝛽0 (Intercept)

  • Estimated by b0 = y-bar - b1(x-bar)
  • Standard Error se(b0) = √MSE (√ (1 ÷ n) + ∑( - )
  • It is usually not meaningful to interpret (mean of the responses when x is 0), because 0 is often beyond the original observation range (extrapolation) or 0 is an impossible value.
  • Hypotheses that test if 𝛽0 is significantly different from 0:
    • :
      • 𝛽0 = 0
    • :
      • 𝛽0 ≠ 0
  • Reject if tobs = (b0) ÷ se (b0) > Critical t(n-2) (or < for negative values) or p < 𝛼
  • Regression through the origin (b0 fixed to 0)? Only when your theory strongly suggests it.

Coefficient 𝛽1 (Slope)

  • Estimated by b1 = rxy (sy / sx)
  • Standard error se(b1) = √(MSE) / √∑(xi - x-bar)2
  • Hypotheses:
    • Is IV a good predictor?
    • Reject H0 if tobs = b1 / se (b1) > Critcal t(n - 2) (or < for negative values) or p < 𝛼
      • In SLR, this t-test result should agree with the omnibus test.
