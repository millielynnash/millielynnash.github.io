Lecture

How/What? Population and Sample

  • Population:
    • All members of a well-defined group (people, animals, parts produced in a factory, districts, schools, etc.)
  • Sample:
    • A subset of a population. Generally, it is too difficult (time-consuming, costly, ineffecient) to collect data on the entire population (census) so we take a sample.
  • We draw a Sample (data) from the Population of research interest and make inference (Generalization) back to population.

Sampling and Making Inference

  1. Population of Interest
  2. Draw Sample
  3. Compute Sample Statistics
  4. Make Inference Back to the Population

Parameter and Statistic

  • Parameter:
    • A characteristic of a population.
  • Statistic:
    • A characteristic of a sample.
  • The parameters in Population are often unknown, but we can estimate them from our sample statistics.
    • Parameters are usually notated by Greek alphabet.
    • Sample statistics are usually notated by English alphabet.

From Population to Sample

  • A good Sample should be:
    • Identical to the characteristics/properties of the population.
    • Representative of a population.
  • This allows the sample to be generalizable to the population.
    • Results will be applicable to other samples drawn from the same population.

Common Sampling Methods

Simple Random Sample (SRS)

  • Every member of the population of interest has an equal probability of being selected.
  • Selection of one unit has no effect on the selection of another unit.
  • Best technique to obtain a representative sample.
  • Use a number table, a compute, a drawing, or a lottery system to generate random numbers.

Stratified Random Sampling

  • Use if one or more characteristics are focal research interest.
  • The goal is to compare groupds (strata). The purpose is to make sure each group is represented in the sample.
  • First divide population into groups (strata). Then a random sample is taken from each group.

Cluster Sampling

  • There are 'naturally occurring' groups in population, usually geographical or organizational.
  • Randomly selected clusters. Then sample all units within each cluster, or random sample with each chosen cluster.
  • Error is smaller if the clusters are similar to each other.

Systematic Sample

  • Every k^th unit is selected from a list, where k=population/desired sample size.
  • Have a list with all units in the population. Then take sample systematically.
  • Make sure the list you are selecting from is random.

Convenience Sampling

  • Choose samples merely based on proximity without considering whether they represent the population or not.
  • Readily available, quick, inexpensive.
  • Can reduce bias by using conveniance sample along with Simple Random Sample (SRS), taking multiple samples, or cross-validation.

Things to Consider

  • Is the sample representative of the population? (Are the findings generalizable?)
  • Sample size?
    • The larger sample does not mean more representative of the population.
    • Required sample size depends on design, confidence level, desired effect size and statistical power. Generally, 30 or more per combinations of factors.
  • How to report:
    • Define your population
    • Describe sampling procedure
    • Indivate desired sample size
    • Indicate actual sample size
    • Provide descriptions of the sample and inferences from the analyses

Variables

  • Variable:
    • Any characteristic of persons or things that is observed to take on different values.
      • Constant:
        • Any characteristic of persons or things that is observed to take on only a single value.
  • Role of variables:
    • Dependent (outcome/target) vs. Independent (predictors/input)
  • Types of variables:
    • Qualitative (Categorical)
    • Quantitative (Numeric)

Qualitative Variables

  • Phenomena or variables that can be used to describe a quality of a person or item.
  • Categorical
    • Dichotomous variables can only take on 2 different values (e.g., yes or no).
  • Example:
    • County of residence, religion, political affiliation, ethnicity
    • Dichotomous:
      • Sex assigned at birth (male or female)
      • Whether a person voted in an election (yes or no).

Quantitative Variables

  • Variables that describe a numerical aspect of characteristic of a person or phenomena.
    • Discrete:
      • A numerical variable that arises from the counting process that can take on only certain values.
    • Continuous:
      • A variable that can take on any value within a certain range, given a precise enough measurement instrument.
  • Example:
    • Discrete:
      • Count of students enrolled in a course, # of clinic visits, agreement level to a survey item
    • Continuous:
      • Distance between two cities, checking account balance, time to rune a marathon, GPA

Levels of Measure

  • Numbers can be assigned to the possible outcomes or responses that can occur for any variable, regardless of how it is classified (i.e., qualitative or quantitative).
  • Measurement:
    • The process of assigning numbers to phenomena in order to represent or distinguish among responses.
      • Represent the features and properties of a variable.
      • Help us understand how to describe and analyze the data.
  • S. S. Stevens (1951) developed 4 types of measurement scales that could be used for assigning these values:
Nominal Variable
  • Usually used for qualitative/categorical variables
  • Numbers mark distinctions between responses, and nothing more. The numbers have no inherent meaning, and the order does not matter at all.
  • Example:
    • County of residence (1=Franklin, 2=Delaware, 3=Union OR 1=Delaware, 2=Union, 3=Franklin)
    • Sex assigned at birth (0=female, 1=male OR 10=female, 15=male)
  • In SPSS, can set value labels and display either data or label.
Ordinal Variables
  • Numbers are assigned to represent ranks or order (usually discrete). Numbers have meaning, but only in terms of distinguishing order, from largest to smallest (or vice versa).
  • Equal distance between two adjacent levels are not assumed.
  • Example:
    • Letter grades:
      • A, B, C, D, E (1-5)
    • Size of a cup of coffee (small, medium, large, 1-3)
    • Socioeconomic status (SES) (low, middle, high, 1-3)
    • Agreement (Likert Scale) (strongly agree, neutral, disagree, strongly disagree, 1-5)
  • Sometimes data needs to be recorded to make interpretation more intuitive
Interval Variable
  • Equal intervals or increments along the scale
  • Interval variables have an arbitrary zero point.
  • Example:
    • Temperature Celsius or Fehrenheit, SAT score, credit score
    • Likert Scale (equal distance assumed):
      • More choices for statistical methods are available for interval data than ordinal data. Decisions might depend on your data.
  • Report range to help readers understand the measure.
Ratio Variable
  • Equal intervals or increments along the scale and contains an absolute and true zero point (absence of the phenomena being measured, data measures from 0 and above).
  • It is possible to compare 2 values and say one is 'twice as large' or 'two-third the size'

Lab

Getting SPSS

  • Statistical Package for the Social Sciences

Opening an SPSS File

  1. Open the SPSS application
  2. File -> New -> Data

Opening an Existing SPSS File

  • File -> Open -> Data -> select the desired file

Opening a File from Excel

  • File -> Open -> Data -> change 'Files of Type' to: Excel -> select the desired file
  • File -> Import Data -> Excel -> select the desired file
    • If the excel file has column titles, click 'Read variable names' from the first row of data.

Data View

  • Each row represents a case (person, school, state, etc.)
  • Each column represents a specific variable

Name

  • Must begin with a letter
  • The rest can contain a combination of numbers and letters
  • Can use periods and underscore, but no other symbol

Type

  • Numeric (most common)
  • Date
  • Dollar
  • String (second most common, only Type where you can use letters)

Width

  • How many characters are allowed inside the cell (string variables)

Decimals

  • How many decimal places are allowed in the cell

Label

  • Where you can further define the variables

Values

  • Used when numbers in the data file represent something

Missing

  • Where you can define certain values to be considered 'missing'

Columns

  • Helps to adjust the size/width of the columns is data view

Measure

  • Nominal:
    • Used for nominal variables such as ID numbers and when numbers represent something
  • Ordinal:
    • Used for ordinal data
  • Scale:
    • Used for ratio and interval data
