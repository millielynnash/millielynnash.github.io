Multiple Linear Regression (MLR)

  • A statistical method analyzing the linear relationship between an interval level dependent variable and two or more independent variables.
AspectDetails
PurposePredict a continuous dependent variable using two or more predictor variables.
Key Assumptions• Linearity: The relationship between predictors and DV is linear (check scatterplots).
• Independence: Residuals should be independent (Durbin–Watson ≈ 2).
• Homoscedasticity: Residuals should display constant variance (assessed via residual plots).
• No Multicollinearity: VIF values should be < 10 and tolerance values > 0.1.
How to Check in SPSS• Create scatterplots for each predictor vs. DV using Graphs → Scatter/Dot.
• In Analyze → Regression → Linear, check the Durbin–Watson statistic and request collinearity diagnostics.
• Examine residual plots for constant variance.
• Review VIF and tolerance values in the coefficients table.
SPSS Steps1. Analyze → Regression → Linear.
2. Enter the DV and all predictor variables.
3. In Statistics, select collinearity diagnostics and the Durbin–Watson statistic.
4. Run the analysis.
Key Output Elements• Model Summary (R, R², Adjusted R²).
• ANOVA table (F and p values for the overall model).
• Coefficients table (B, Beta, t, p values, VIF).
• Durbin–Watson statistic.
Criteria for Assumptions Met• Durbin–Watson ≈ 2.
• VIF < 10; Tolerance > 0.1.
• Scatterplots show a linear trend and residuals display no funnel shape (homoscedasticity).
APA Reporting Example"A multiple linear regression analysis was performed to predict exam scores from study hours and attendance. The overall model was significant, F(3, 56) = 15.41, p < .001, accounting for 61% of the variance (R² = 0.61). Study hours (β = 0.45, t(56) = 6.83, p < .001) and attendance (β = -0.28, t(56) = -3.78, p = .002) were significant predictors. The Durbin–Watson statistic was 1.98, and VIF values were below 2, indicating no multicollinearity concerns."

Purpose and Use Case

  • Predicts a continuous dependent variable using multiple independent variables.
  • Example: Predicting exam scores based on study hours and attendance.

Assumptions & Their Checks

AssumptionRequirementHow to Check in SPSS
LinearityThe relationship between predictors and DV is linear.Scatterplots of each predictor vs. DV.
Independence of ObservationsResiduals should be independent.Check the Durbin–Watson statistic (ideal ≈ 2).
HomoscedasticityResiduals have constant variance.Inspect residual scatterplots.
MulticollinearityPredictors should not be too highly correlated.Check VIF values (VIF < 10 is acceptable).

Steps to Conduct in SPSS

  1. Analyze → Regression → Linear.
  2. Move your dependent variable to the Dependent box and predictors to the Independent(s) box.
  3. Click Statistics and select Collinearity diagnostics and Durbin–Watson.
  4. Click OK.

More Detailed How-To in SPSS

When to Use MLR?

  • MLR is used when you want to understand the relationship between one Dependent Variable and two or more Independent Variable. It is commonly applied in scenarios such as:
    • Predicting Outcomes
      • For example, predicting house prices based on square footage, number of bedrooms, and location.
    • Understanding Relationships
      • For example, analyzing how factors like advertising spend, produce price, and customer satisfaction influence sales.
    • Controlling for Confounding Variables
      • For example, studying the effect of education level on income while controlling for age and work experience.

Assumptions of MLR

Multicollinearity

  • In Multiple Linear Regression (MLR) multicollinearity is said to exist when two or more of the independent variables are themselves highly correlated, a situation that makes it difficult to establish the effect each has on the dependent variable.
  • How to Check
    • Calculate the Variance Inflation Factor (VIF).
      • Values above 5-10 indicate multicollinearity.
  • Example
    • Check if square footage and number of bedrooms are highly correlated.
  • Solution
    • Remove or combine correlated variables.

Testing Linearity

  • The relationship between the Dependent Variable and each Independent Variable is linear.
  • How to Check
    • Use scatterplots to visualize relationships.
    • Click Graphs > Legacy Dialogs > Scatter/Dot.
    • Click Simple Scatter. Click Define.
    • Move your Dependent Variable (Outcome) to the Y-Axis. Move your Independent Variable (Predictor) to the X-Axis
    • If the scatterplot is approximately linear, the assumption is met.
  • Example
    • Plot house price against square footage to see if the relationship linear.
  • Solution
    • If not linear, apply transformations (e.g., log, square root)

Testing Homoscedasticity

The residuals (errors) have constant Variance across all levels of Independent Variable s.

  • How to Check
    • Plot residuals vs. Predicted values
    • Look for a random scatter.
    • In SPSS:
      • Request Linear Regression Plots.
      • Y > ZPred
      • X > Z Resid
      • If the scatterplot shows no overall pattern, the assumption is met.
  • Example
    • Check if the spread of residuals is consistent across predicted house prices.
  • Solution
    • Apply transformations or use weighted least squares.

Testing for Normality

An assessment of the normality of data is a prerequisite for many statistical tests because normal data is an underlying assumption in parametric testing.

Methods of Assessing Normality

  • SPSS allows you to test all of these procedures within Explore command. The Explore command can be used in isolation if you are testing normality in one group or splitting your dataset into one or more groups. If you have 2 or more categorical, independent variables, the Explore command on its own is not enough and you will have to use the Split File command also.
Procedure for None or One Grouping Variables
  1. Click Analyze -> Descriptives -> Explore
  2. Transfer the variable that needs to be tested for normality into the Dependent List box.
  3. Click on Statistics. Leave Descriptives selected.
  4. Click the Plots. Select Factor Levels Together under Boxplots. Select Stem-and-Leaf under Descriptive. Select Normality Plots with Tests.
Output
  • When testing for normality, we are mainly interested in the Tests of Normality table and the Normal Q-Q Plots, our numerical and graphical methods to test for the normality of data, respectively.
Shapiro-Wilk Test of Normality

  • The above table presents the results from two well-known tests of normality, namely the Kolmogorov-Smirnov Test and the Shapiro-Wilk. The Shapiro-Wilk Test is more appropriate for small sample sizes (<50 samples), but can also handle sample sizes as large as 2000.
  • If the Sig. value of the Shapiro-Wilk Test is greater than p = .05, the data is normal. If it is below 0.05, the data significantly deviate from a normal distribution.
Normal P-P Plot

  • In order to determine normality graphically, we can use the output of a normal P-P plot. If the data are normally distributed, the data points will be close to the diagonal line. If the data points stray from the line in an obvious non-linear fashion, the data are not normally distributed.
  • If the points are generally along the diagonal, the assumption is met.
  • If you are at all unsure of being able to correctly interpret the graph, rely on the numeric methods instead because it can take a fair bit of experience to correctly judge the normality of data based on plots.
Histogram

  • If the histogram is approximately normal, the assumption is met.
  • The distribution curve is normal/bell shaped.
Skewness and Kurtosis
  • The assumption is met if the skewness and kurtosis is between +2 and -2.
Procedure When There Are Two or More Independent Variables
  • The Explore command on its own cannot separate the dependent variable into groups based on not one but two or more independent variables. However, we can perform this feat by using the Split File command.
  1. Click Data -> Split File
  2. Click Organize Output by Groups. Transfer the independent variables you wish to categorize the dependent variable on into the Groups Based On box.
  3. Proceed as above.

Testing for Extreme Outliers

Utilize Cook's Distance under the 'Residuals Statistics' Table

  • If the maximum Cook's distance is below 1.0, then the assumption is met.
  • Other tests may give a different cutoff point, but for our purposes we will use 1.0.

Testing the Independence of Error

Observations are independent of each other (no Autocorrelation)

  • How to Check
    • Use the Durbin-Watson test.
      • If the Durbin-Watson test falls within the range of 1.5 - 2.5, the assumption is met.
  • Examples
      • For this example, we'll say 1.482 is close enough to 1.5 and the assumption is met.
    • Ensure house prices are not influenced by neighboring houses.
  • Solutions
    • Use Time-Series Design if data is Autocorrelation.

SPSS

  1. Analyze -> Regression -> Linear
  2. Move Dependent Variable to Dependent. Move Independent Variable (s) to the Independent (s).
    • In Statistics, select Estimates (Under Regression Coefficients), Model Fit, Descriptives, and Durbin-Watson (Under Residuals). Click Continue.
    • In Plots, select Histogram and Normal Probability Plot (Under Standardized Residual Plots). Move ZPRED under Y. Move ZRESID to X. Click Continue.
    • In Save, select Cook's (Under Distances). Click Continue.
    • Click OK.

Procedure for Multiple Regression (Statistics)

  • Include Model Fit, R-squared change, Descriptives, Part and Partial Correlations, and Collinearity Diagnostics
  • Confidence intervals are not necessary but can be helpful.

Procedure for Multiple Regression (Save)

  • You can save residuals as their own variables to detect multivariate outliers.
  • Predicted values tell you what SPSS thinks the y value should be based on the x-values of the case.
  • Residual values are how far off SPSS was
  • Mahalanobis distance (Mahal Distance) tells you how far away a case is from the 'cloud' of other values.
