Simple Linear Regression (SLR)

  • Also known as Ordinary Least Squares (OLS)
  • Correlation does not imply causation. With proper literature review and research design, however we can investigate how one variable (x or IV) explains the variability (variance) in the other variable (y or DV). If x is indeed a significant predictor of y, we can use x to predict y.
  • Still a bivariate analysis, but with a different goal in mind. For example, the school district might want to use GPA to explain the variable in ACT scores and use GPA to predict ACT.
  • Here we have only one predictor (IV) in the model: Simple Linear Regression (SLR).
  • Regression means you are trying to find a pattern between x and y. You are using x to predict y. We are only using one.

Related Terms, but Not Necessarily the Same

ExplanationPrediction
- Explanation refers to 'explained variance.'
- If you can explain why differences exist (among people, over time, etc.), you have learned a lot about the phenomena of interest.
- Explanation involves theory of how relationships arise: Contributes to theory or intervention development		- Prediction may simply want an adequate model of the outcome.
- May not be related to theory (e.g., ice cream sales predict shark attacks).

Model Summary: Variance in DV Explained

  • Total variability (variance) in DV is partitioned into two parts:
    • What we can explain by the model (regression)
    • What we still cannot explain (error)
  • Regression sum of squares (SSR)
  • Error Sum of Squares (SSE)
  • Total Sum of Squares (SSTO)
  • SSTO = SSR + SSE
  • SSR, SSE, and SSTO are available in the ANOVA table

Example: Regression rating on length (Model Summary)

ANOVA Table and the Omnibus Test

  • Omnibus:
    • Comprising several items
    • "Congress passed an omnibus anticrime package."
  • The omnibus test is used to evaluate the significance of overall regression model. Statistics used to conduct the omnibus test are available in the ANOVA table.
  • Many different (equivalent) ways to represent the omnibus test, but they are asking the same question:
    • Is the regression model good enough to be used to explain/predict DV?
  • Since we are comparing variances (SSR and SSE), the sample statistics is compared to a critical F statistics, given the significance level (usually Œ±=. 05). General idea:
    • Large observed F means much variance in DV can be explained by the model (desired)!

Omnibus Test Significance (SLR)

  • Compare Fobs to the critical F(1, n -2)
    • Reject H0 if Fobs > F(1, n -2)
  • Alternatively, reject H0 if p < Œ±
  • For APA style writing, make sure you report the F statistic along with both degrees of freedom and the p-value.
Example: Regressing rating on length (ANOVA table)
  • SSR = 4.918
  • SSE = 43.609
  • Fobs = MSR / MSE = (SSR / 1) / (SSE / n - 2) = (4.918 / 1) / (43.609 / 98) = 11.052
  • Since p = .001, reject H0 (significant model/predictor)
  • Report F (1, 98) = 11.052, p = .001
  • Can also check R2 = (SSR) / (SSTO) = 4.918 / 48.528 = .101
Example: Regressing rating on year (ANOVA table)
  • SSR = 1.057
  • SSE = 47.471
  • Fobs = MSR / MSE = (SSR / 1) / (SSE / n - 2) = (1.057 / 1) / (47.471 / 98) = 2.182
  • Since p = .143, cannot reject H0 (model/predictor not significant)
  • Report F (1, 98) = 2.182, p = .143
  • Can also check R2 = (SSR) / (SSTO) = 1.057 / 48.528 = .022
