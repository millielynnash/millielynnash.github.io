Bivariate Regression

  • The regression of one variable on another.

Assumptions of Regression

  1. Assumption of Normality of Error (Testing for Normality)
  2. Assumption of Homoscedasticity (Testing Homoscedasticity)
  3. Assumption of Independence of Error (Testing the Independence of Error)
  4. Assumption of No Extreme Outliers (Testing for Extreme Outliers)
  5. Assumption of Linearity (Testing Linearity)
  • Note: The assumption for any statistical analysis is about the Outcome, not the Predictor .

Fundamentals of Bivariate Regression

  • To use a bivariate regression model, identify two types of variables:
    • Predictor, or known variable (x)
    • Outcome, or to-be-predicted variable (y)
  • Regression analysis provides an equation that corresponds to a line that best predicts y scores from x scores.
  • The model Population Parameter will be designated with a Greek letter (β).
  • The estimate of the population parameter, which we calculate using our sample data, is designated by the Arabic letter b.

Simple Model vs. Predictive Model

  • We can predict a person's score on an outcome (dependent) variable (Y-Hat) based on what we know about that person's score on a predictor (independent) variable (x).
  • If variables x and y are related to each other, then adding x to our model improves prediction and produces less error (SSE) than the simple model generated.

Bivariate Regression

  • You can use bivariate regression to answer the following questions about the pattern of data points and the significance of a linear equation:
    1. Is a pattern evident in a set of data points?
    2. Which equation of a straight line can best describe this pattern?
    3. Are the predictions made from this equation significant?
  • More questions using Bivariate Regression:
    1. What is the nature of the relation between x and y?
    2. Is b statistically significant? What is the Confidence Interval for b.
    3. What is the effect size?
    4. How is the y variance partitioned between explained and unexplained?
    5. Do plots of residuals indicate violations of assumptions for bivariate linear regression?

Centering the Predictor Variable
  • For the simple mean-only model, 𝛽0, was a constant equal to the mean of Y.
  • Since the mean of Y was 6, 𝛽0 = 6.
  • The intercept, 𝛽0, is the predicted value of Y when X is zero.
  • Thus, if X has no meaningful value of 0, the estimated intercept is not informative.
  • Centering the predictor of X to its mean can generate a better estimate because then the estimated 𝛽0 is now the predictor value of Y when the predictor X is its mean.
  • Think about the 1 st year college CPA as the outcome (Y) predicted by the SAT score (X).
  • Y = 10.268 - .776 X + 𝜀
  • Y = 6 - .776 X' + 𝜀
  • This new regression equation shows the exact same relationship between Y and X' as we found for Y and X.
  • If you calculate the error of this regression model, you will get the exact same SSEA
