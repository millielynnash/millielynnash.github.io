Data = Model + Error

The Data

  • Data:
    • Values derived from scientific experiments
    • Data can refer to countless variables, scores, and sources of information
    • To understand a phenomenon, a behavioral scientist must first operationally define it as a measurable variable.
    • Common descriptive statistics are the mean and standard deviation.

The Model

  • Prediction is one way to demonstrate understanding of something.
  • A model that predicts a phenomenon well implies that we understand that phenomenon.
  • An initial model that predicts the mean for every subject serves as the foundation on which subsequent models are based.
  • Y = 𝛽0 + 𝜀 (Simplest model that we use)
    • The single predictive component of the model () is designated as 𝛽0.
    • Comparing the actual scores (Y) and the predicted scores reveals how much error exists in this simple model.
    • Rarely would all of the scores on the dependent variable equal the mean.
    • We study the error in our model in hopes that we can reduce the error to improve our model.
    • To reduce the error, we must be able to measure error.

Conceptual vs. Computational Formula

  • Conceptual Formula
    • SSE = ∑ (Yi - Y-Hati)2
    • It captures the concept of error by squaring and summing the differences between predicted and actual scores.
    • It is a bit clumsy, especially when the mean is not an integer.
  • Computational Formula
    • SSE = ∑Y2 - (∑Y)2 / N)
    • This is more convenient to compute SSE compared to the conceptual formula.

Variance and Standard Deviation (SD)

  • Variance (S2) and standard deviation (S) describe the variation of scores around a mean.
  • Variance is SSE divided by the number of scores minus 1:
    • SSE / (N - 1)
    • Variance = Sy2 = (∑Y2 - (∑Y)2 / N)) / (N - 1)
  • Since we squared the variable's scores to make them positive, we take the square root of the variance to return to the variable's score to return to the variable's origina; scale.
  • The square root of the variance is called the standard deviation (S)

Hypothesis Testing

  • We want to know whether the mean is statistically significantly different from 0.
  • We use the one-sample t-test to examine whether μ = 0
  • Hypothesis testing is a decision-making process where two-possible decisions are considered.
    • H0 (Null or Statistical Hypothesis)
    • H1 (Alternative or Research Hypothesis)
  • Definition
    • A method for testing a hypothesis about a Parameter or Population, using data measured in a sample.
  • Purpose
    • To test claims or ideas about a group or Population.
  • Goal
    • To determine the likelihood that a Population Parameter, such as the Mean, is likely to be true.
  • Four Steps of hypothesis testing
    1. State the null and alternative hypothesis
      • Null Hypothesis (H0)
        • Statement about the population parameter that is assumed to be true.
        • Center orange line.
        • H0 = 3
      • Alternative Hypothesis (H1)
        • Statement that contradicts the null hypothesis.
        • Circled tails.
        • H1 ≠ 3
    2. Set the criteria for a decision
      • State the level of significance
      • Criterion of judgement upon which a decision is made regarding the value stated in a null hypothesis.
      • Typically the level is set at 5% in behavioral research studies.
    3. Compute the test statistic
      • Test statistic
        • Z-score
          • How far a sample outcome is from the value stated in a null hypothesis
          • z = (x - M) / SD
          • zobt = (M - μ) / σM
          • The z-score can be used to make a decision regarding the null hypothesis.
          • The larger the value, the further a sample mean deviates from the population mean.
        • T-score
          • tobt = (M - μ) / σM
    4. Make a decision
      • Based on the probability of obtaining a sample mean, given that the value stated in the null is true (represented by p-value).
      • Reject H0
        • The sample mean is associated with a low probability of occurrence.
      • Retain (fail to reject) H0
        • The sample mean is associated with a high probability of occurrence.
      • Statistical significance
        • If z-statistic falls within 2 SE of M, sample mean is 'normal' (p > 0.05)
        • If z-statistic falls outside 2 SE of M, sample mean is 'extreme' (p < 0.05)

Types of Error

Reality vs. DecisionFail to Reject H0Reject H0
H0 is true (chance)Life goes on as usual, status quoFalsely reject the null hypothesis. False positive. Type I Error (𝛼)
H0 is false (theory)Falsely accept the null hypothesis. False negative. Type II Error (𝛽)Correct decision, theory confirmed! Statistical power = (1 - 𝛽)

Reading the t-Table

  • You need 3 pieces of information to use the t-table
    1. Sample size
    2. α-level
    3. One-tailed or two-tailed?
  • Reading a t-table is different from reading a z-table.
  • You calculate t-value from your data and find the critical t-value from the table.
  • If the calculated t is greater than the critical t, you reject the H0.

Directional vs. Non-Directional Hypotheses

  • Specifying the direction (positive/negative, greater/lesser, etc.) during hypothesis testing
  • Non-Directional
    • Example
      • Is there a mean difference in motivation scores between students in a persistence-based curriculum as compared to the control curriculum?
      • H0: µpersistence motivation = µcontrol motivation
      • H1: µ persistence motivation ≠ µcontrol motivation
  • Directional:
    • Example
      • Do students in a persistence-based curriculum report higher mean motivation scores than those in a control curriculum?
      • H0: µpersistence motivation = µcontrol motivation
      • H1: µ persistence motivation > µcontrol motivation

Other Notes

  • 'Fail to reject the null' vs. 'Confirm [accept/support] the null'
    • 'Fail to reject the null'
      • Cons
        • Double-negative, an editor at the communications will tell you that this is badly written, but there is a technical difference which matters. We are not determining whether we accept/confirm/support, but we are determining whether we reject the null.
      • Pros
        • Statistically accurate.
  • Tip:
    • Avoid using terms like 'confirm' (alternative: results provide support for the alternative hypothesis)
  • Null Findings
    • Pros
      • Important for being good stewards of numbers to make inferences about people.
    • Cons
      • Bad for your academic career
  • What is 'Significant'?
    • Beware of terms like 'approaching significance'
    • .05 - this is a kind of arbitrary number
